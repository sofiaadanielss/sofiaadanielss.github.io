Haskell es un lenguaje de programación funcional, mientras que C es un lenguaje imperativo. En C se describe paso a paso cómo resolver un problema; en Haskell simplemente se describe qué se quiere obtener. Esto cambia profundamente el estilo de programación.

C vs. Haskell

C Haskell
Variables mutables Valores inmutables
Usa ciclos imperativos (for, while) Usa recursión o funciones como map, filter, fold
Funciones con efectos secundarios Funciones puras
Manejo manual de memoria Garbage Collector

Tipos en Haskell

suma :: Int -> Int -> Int
suma a b = a + b

  • Regresa un Int
